Detailed Description

Helper class to get the consumer friendly device name.

String device = Devices.getDeviceName();

The above code will get the device name as specified in the supported device list that is maintained by Google.

See: https://support.google.com/googleplay/answer/1727131

final String [] com.gtfp.workingmemory.utils.Devices.SUPPORTED_DEVICES
staticprivate

This array is using the list of supported devices on Google Play.

Any device that does not have both the retail name and marketing name is not included.

See the following URL for the PDF that this list was derived from: https://support.google.com/googleplay/answer/1727131

Each String uses a vertical line to separate the manufacturer, name, device, and model:

"Retail Branding|Marketing Name|Device|Model"
